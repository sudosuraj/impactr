Una raccolta di progetti della comunità costruiti su Impactr.

:::note
Vuoi aggiungere il tuo progetto legato a Impactr a questa lista? Apri una PR.
:::

Puoi anche dare un'occhiata a [awesome-impactr](https://github.com/awesome-impactr/awesome-impactr) e [impactr.cafe](https://impactr.cafe), una comunità che aggrega ecosistema e community.
