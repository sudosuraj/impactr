Una colección de proyectos comunitarios construidos en Impactr.

:::note
¿Quieres agregar tu proyecto relacionado con Impactr a esta lista? Envíe un PR.

También puedes consultar [awesome-impactr](https://github.com/awesome-impactr/awesome-impactr) y [impactr.cafe](https://impactr.cafe), una comunidad que agrega el ecosistema y la comunidad.
