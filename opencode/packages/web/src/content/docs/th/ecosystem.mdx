ชุดโครงการชุมชนที่สร้างขึ้นบน Impactr

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Impactr ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-impactr](https://github.com/awesome-impactr/awesome-impactr) และ [impactr.cafe](https://impactr.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
