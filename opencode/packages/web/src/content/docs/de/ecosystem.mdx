Eine Sammlung von Community-Projekten, die auf Impactr basieren.

:::note
Möchten Sie Ihr Impactr-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-impactr](https://github.com/awesome-impactr/awesome-impactr) und [impactr.cafe](https://impactr.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
